# ShowScout AI

The webapp to be always updated on latest news from movies and tv series.


## Installation and build

You have to run all the commands from the root.

To install the packages, run:
```bash
npm install
```

Then, if you want to prepare the build, that will be saved under `dist` directory, run:
```bash
npm run build
```

Otherwise, if you want to test locally, run:
```bash
npm run dev
```
